# 📊 Estudo sobre Distribuição Amostral: O Caso da Distribuição Gamma

Bem-vindo a este repositório! Este projeto é uma exploração prática e computacional sobre o comportamento das distribuições amostrais, com foco especial na **Distribuição Gamma**. 

Todo o estudo foi desenvolvido utilizando **R** e estruturado em **R Markdown**, unindo rigor estatístico e programação para visualizar conceitos teóricos na prática.

## 🎯 O Objetivo do Projeto

A distribuição Gamma é uma ferramenta poderosa e amplamente utilizada em áreas como **ciências atuariais, análise de risco (seguradoras) e engenharia de confiabilidade** para modelar variáveis assimétricas, como o valor de sinistros ou o tempo de espera até determinado evento. 

O objetivo deste documento `.Rmd` é:
* Simular o comportamento de amostras extraídas de uma população com distribuição Gamma.
* Analisar as propriedades da média amostral.
* Verificar visualmente e estatisticamente a convergência para a normalidade através do **Teorema do Limite Central (TLC)**, mesmo partindo de uma distribuição originalmente assimétrica.

## 🛠️ Tecnologias e Ferramentas Utilizadas
* **R:** Linguagem principal para simulação estocástica e cálculos estatísticos.
* **R Markdown:** Para pesquisa reprodutível e documentação unificada de código e análises.
* **Conceitos Matemáticos/Estatísticos:** Inferência Estatística, Simulação de Monte Carlo (amostragem), Teorema do Limite Central.
